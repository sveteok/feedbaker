Feedbaker Documentation

Welcome to the **Feedbaker Developer Documentation** - your guide to setting up, using, and extending the Feedbaker feedback management system.

Feedbaker is a lightweight platform that helps developers and product owners **collect, manage, and summarize user feedback** for their websites and web apps.
